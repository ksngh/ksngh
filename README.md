## 안녕하세요, 백엔드 개발자 김성호입니다

문제를 끝까지 책임지고 해결하는 것을 중요하게 생각하며,  
목표를 향한 꾸준한 학습과 개선을 실천하는 **백엔드 개발자**입니다.

기획 단계부터 아키텍처 설계, 개발, 배포까지  
프로젝트 전 과정에 **주도적으로 참여**해 왔으며,  
배려와 협업을 바탕으로 팀에 긍정적인 에너지를 더하는 동료가 되고자 합니다.

---

## 기술 스택

- **Backend**
  - Java, Spring Boot

- **Search / Cache**
  - Elasticsearch (최근·인기 검색, 검색 품질 개선)
  - Redis

- **Data**
  - RDBMS (MariaDB, PostgreSQL)

- **Infra / DevOps**
  - Docker
  - AWS
  - CI/CD 구축 및 운영

---

## 주요 관심 영역

- Elasticsearch 기반 검색 기능 설계 및 품질 개선
  - Analyzer 튜닝
  - Precision / Recall 기반 검색 실험
- ReBAC(Relationship-Based Access Control) 기반 서비스 설계
- 테스트 주도 개발과 안정적인 백엔드 아키텍처
- 운영 환경을 고려한 성능 최적화 및 장애 대응
